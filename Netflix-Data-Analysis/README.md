# 🎬 Netflix Data Explorer

Welcome to the **Netflix Data Explorer**!
This project dives into the Netflix Movies & TV Shows dataset to uncover interesting trends about content types, genres, countries, and how Netflix has grown over time.

Think of this as a **mini data detective project** 🔍 powered by Python!

## 🚀 Project Goal

Answer key questions about Netflix content:

* 🎥 How many **Movies vs TV Shows** are on Netflix?
* 🌍 Which **countries produce the most content**?
* 🎭 What are the **most popular genres**?
* 📈 How has **Netflix content grown over time**?

## 🛠️ Tech Stack

* 🐍 Python
* 📊 Pandas
* 📉 Matplotlib
* 📓 Jupyter Notebook / Google Colab

## 🔍 Key Insights

Some interesting findings from the analysis:

✔ Netflix hosts **more movies than TV shows**
✔ The **United States produces the most content**
✔ **Drama and International content** dominate the platform
✔ Netflix content **rapidly increased after 2015**
